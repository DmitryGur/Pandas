## 📌 Цели и задачи проекта

Проект представляет собой анализ датасета с резюме HeadHunter по городам РФ.  
Цель — выявить ключевые факторы, влияющие на зарплатные ожидания соискателей, и подготовить данные для построения прогнозных моделей.

**Основные задачи:**
- Изучение данных
- Преобразование и очистка данных
- Создание бинарных признаков (занятость, график, готовность к переезду)
- Визуализация распределений и зависимостей
- Выявление и удаление выбросов
- Анализ влияния образования, города, возраста, мобильности на зарплату
